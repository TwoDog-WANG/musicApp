## 使用
1. 歌曲播放、上下一首、随机顺序单曲、音量调节
2. 小窗口点击小点，会有大小窗口切换，最小化，关闭三个按钮
3. 大窗口左上角有设置按钮，现阶段只有设置文件位置，第一次载入或者重新载入一个新的文件夹内的歌曲需要一段时间，请耐心等待
4. 支持歌曲名、作者、时长排序，支持歌曲名、作者搜索
5. 支持快捷键：
    1. 上下左右方向键：加大减小音量，上下一首
    2. alt+c：切换播放模式
    3. 空格：暂停、播放
6. 全局快捷键：在快捷键的基础上增加ctrl
